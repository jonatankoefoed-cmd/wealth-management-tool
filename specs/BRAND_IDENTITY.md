# BRAND_IDENTITY.md
Personal Wealth Management Web App
Version 1.0

## 1) Brand essence
Keywords:
- Calm
- Minimal
- Premium
- Analytical
- Soft geometry
- High trust

Tone of product:
- Neutral and precise
- No hype language
- Short labels
- Units everywhere (DKK, %, per month)
- No emojis

## 2) Visual style direction
Primary inspiration:
- Clean SaaS dashboards with warm neutral surfaces, soft highlights, and restrained color.
- One strong accent per screen. Everything else stays neutral.

Non negotiables:
- Rounded corners across the entire UI, including charts.
- Subtle shadows only.
- High whitespace and clear hierarchy.
- Real icons only, consistent set, no emojis.

## 3) Color strategy
### 3.1 Base
The app should feel warm and calm, not “bank blue”.
Base neutrals are slightly warm (cream/stone), not pure grey.

### 3.2 Accent philosophy
- Primary accent is a calm Sage.
- Amber is reserved for highlights and small emphasis.
- Orange is reserved for alerts or key callouts, never as a default theme.

Rule:
- At most one strong accent color is used as the dominant accent on a screen.
- Secondary colors are used at low opacity only.

## 4) Typography
Font:
- Inter or system font stack with tabular numerals support.

Rules:
- Default numbers use tabular numerals.
- Medium weight for headings, regular for body.
- No overly bold typography.

## 5) Layout and components
### 5.1 Shell
- Left sidebar navigation, compact, high contrast labels.
- Main content uses a max width and consistent gutters.
- Top header shows page title + primary action.

### 5.2 Cards
- Cards are the primary container.
- Card style: warm surface, rounded-lg, soft shadow, thin border.

### 5.3 Inputs
- Inputs are rounded, quiet borders, strong focus ring in accent.
- Units and frequency are explicit.

### 5.4 Tables
- Numeric columns right aligned.
- Minimal separators.
- Row hover is subtle.

## 6) Charts
Rules:
- One highlighted series (accent).
- Other series are muted.
- Gridlines are extremely subtle.
- Rounded chart containers.
- Tooltips are compact and clean.

## 7) Motion
- Subtle transitions only (150 to 220ms).
- No bounce, no playful effects.
- Motion is used to clarify state changes.

## 8) Iconography
- Single icon library only.
- Rounded, thin stroke.
- Default icon color is secondary text, accent only for primary actions.

## 9) Assets required
Minimum set for MVP:
- Brand mark (SVG)
- Wordmark (SVG)
- Lockup (SVG)
- Favicon (SVG + 16 + 32 PNG)
- App icon (1024 PNG + downscales)
- 3 empty state illustrations (SVG, neutral, minimal)

## 10) Definition of done
A screen is brand compliant if:
1) It uses warm neutral base surfaces.
2) It uses one dominant accent only.
3) It has soft corners and subtle shadows.
4) Icons are real icons only.
5) Derived numbers show calculation steps.
6) It is readable and not dense.
